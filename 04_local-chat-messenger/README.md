# Local Chat Messenger
同一マシン上でのプロセス間通信を理解するために、Pythonのsocketモジュールを用いて実装したクライアント・サーバ型アプリケーションです。

### 学習したこと
- クライアント・サーバモデルの基本構造
- socket通信の基本（bind → listen → accept → recv / send ）
- socketモジュールを用いたプロセス間通信
- `AF_UNIX`による同一マシン内通信
- バイト列（`recv()の返り値`）と文字列の変換（`decode()`）
- 外部ライブラリの導入と利用

## 機能
クライアントから送信されたメッセージに対し、サーバがfakerを用いてランダムな名前を生成し、返信します。

## 工夫
- exitコマンドによる通信終了制御を実装しました。
- ユーザーが正しい入力を理解しやすいように、都度、標準出力で案内を表示するようにしました。

## 実行方法
1. サーバの起動
```
python3 src/server.py
```

2. 別ターミナルでクライアントを起動
```
python3 src/client.py
```

3. クライアントでは、`exit`を入力するとで通信を終了することができます。

4. `Control` + `C` の割り込みで終了することができます。

## 動作環境
- Python 3.8以降
- ターミナル

## 使用技術
