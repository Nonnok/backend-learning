# Guess The Number Game
Pythonの標準入力・出力の理解を目的として作成した数字当てゲームです。

### 学習したこと
- Pythonの書き方
- 標準入力（ `input()` ）・標準出力（ `print()` ）の基本操作
- ループ処理（ `for` ）
- 共通処理の関数化

## 機能
- ユーザーは最小値・最大値・試行回数を指定することができます。
- ユーザーが指定した最小値と最大値の範囲内で乱数が生成されます
- ユーザーは指定した試行回数だけ、数字を予想して入力することができます。

## 工夫
- 共通処理を関数化しました。
- ユーザー入力のバリデーションを考えうる限り細かく実装しました。
- ユーザーが正しい入力を理解しやすいように、都度、標準出力で案内を表示するようにしました。

## 実行方法
```
python3 guess-the-number-game.py
```

## 動作環境
- Python 3.x
- ターミナル
