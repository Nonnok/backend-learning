# Backend Learning
このリポジトリは、プログラミング学習サイト「RecursionCS」のバックエンドプロジェクトコースで出題された課題の成果物をまとめたものです。

## 目的
バックエンドの基礎技術を習得すること。

## 課題一覧
<dl>
  <dt>
    <strong> - Guess the number game</strong>
  </dt>
  <dd>
    標準入力・出力の理解を目的とした数字当てゲームです。
  </dd>
  <dt>
    <strong> - File Manipulator Program</strong>
  </dt>
  <dd>
    OSとやり取りしてファイルシステムを操作するプログラムです。
  </dd>
  <dt>
    <strong> - Markdown to HTML Converter</strong>
  </dt>
  <dd>
    外部ライブラリを用いたファイル形式変換プログラムです。
  </dd>
  <dt>
    <strong> - Local Chat Messenger</strong>
  </dt>
  <dd>
    同一マシン上でプロセス間通信を行うプログラムです。クライアントから受信したメッセージに対して、ダミーデータを返信します。
  </dd>
</dl>
